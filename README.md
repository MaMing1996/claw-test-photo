# claw-test-photo

照片分类管理项目（MVP）

## 核心功能

### 1) 按时间分类
- 维度：日 / 月 / 年
- 选择任意日/月/年后，展示该时间范围内的所有照片
- 每个时间分组封面：取该分组中按拍摄时间倒序的最新 3 张，重叠展示

### 2) 按地点分类
- 从照片元数据（EXIF GPS）中解析地点
- 按国家分组展示
- 没有地点信息的照片统一归类为 `Unknown`

## 数据来源（MVP 假设）
- 本地照片目录扫描
- 元数据字段：
  - `capturedAt`（拍摄时间，优先 EXIF，退化到文件时间）
  - `country`（由 GPS 逆地理得到；无则 Unknown）

## 里程碑
1. 导入照片元数据（时间 + GPS）
2. 完成时间分组（日/月/年）
3. 完成国家分组（含 Unknown）
4. 分组封面重叠展示（最新 3 张）
5. 基础 UI（列表 + 分组详情）

## 开发计划（建议）
- Phase 1: CLI 原型（先验证分组逻辑）
- Phase 2: Web UI（分组浏览 + 封面）
- Phase 3: 性能优化（索引/缓存）
